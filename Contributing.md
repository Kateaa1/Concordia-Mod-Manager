### Contributing
All code must be in C++ (unless you have a good reason for it not to be)
All code must be human written
** ALL C++ FILES MUST USE THE `.cc` EXTENTION FOR SOURCE AND `.hh` EXTENTION FOR HEADERS **
Please only use multi-line comments if your comment actually spans multiple lines
Insert a new line when the current line reaches the end of your screen ending on the end of a word
Please document functions with comments above where it is defined
Put stuff in new files where it makes sense, main should be initiisation and basic function calls
Please compile and check your code works before opening a PR
